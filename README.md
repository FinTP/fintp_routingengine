fintp_routingengine
===================

Performs basic ETL routines on messages.

Contributing

Requirements
------------
- Boost
- Xerces-C
- Xalan-C
- ZipArchive
- Libtiff
- pthread
- **fintp_utils**
- **fintp_log**
- **fintp_transport**
- **fintp_udal**
- **fintp_base**
